# examples/
Read-only fictional samples — one per private entity type — so you can see the
file format without exposing real data. Copy one into the matching real folder
(`problems/`, `decisions/`, `reflections/`, `systems/`) and edit, or just use the
slash commands. These use sentinel IDs (`*-20000101-*`) and `-example-` filenames
so they are unmistakably not real entries.
