# precedents/
A public, shared library of **how problems have been solved** (`PRE-YYYYMMDD-NN-*.md`).
New problems are matched against these by `subjects`/`problem_types`/`tags`. Grow it
with `/add-precedent`, or by distilling a de-identified precedent from a resolved
problem via `/outcome`. Public/tracked — never include personal identifying details.
See `../SCHEMA.md`.
